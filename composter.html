Cryptography Puzzle Game Website

  • GitHub
  • React Website

For COMP 426: Modern Web Development, I was tasked with creating a website from scratch with the freedom to create almost anything I wanted. Given this freedom, I decided to recreate one of my favorite games; a puzzle game where players are trying to decode an encrypted quote, typically from a well known individual such as Alan Turing or Benjamin Frankin.

To create this React website, I used two different APIs. The first API is a random quote generator to actually get a quote. The second API is to get information from Wikipedia and provide the player with a short summary of the person the quote is by incase they are unfamilair with the individual. To encrypt the quote, I used a monoalphabetic subsitution cipher. This means that each letter is transformed into a different letter. For example, every "a" could become a "j" and every "j" could become a "c". Although this encryption is not secure whatsoever, it allows a player to use letter frequency analysis and other techniques to discover the plaintext of the message without any complicated calcualtions.

Cryptography Puzzle Game Website Features

Features include:

  • React framework
  • RESTful APIs
  • Custom CSS
  • Toggleable light and dark themes
  • Custom quote encryption

As my first endeavor in creating a website from scratch, I realized how powerful and versatile React can be. Being able to create components and reuse code is such a valuable tool and saves large amount of times, especially when creating larger projects. Although frustrating at first, I learned how to work with APIs to get and send data and will be exploring what other APIs can do in the future.
